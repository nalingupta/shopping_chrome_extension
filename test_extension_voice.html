Testing Voice Input in the Shopping Assistant Extension

Step 1: Load the Extension

Make sure the Shopping Assistant extension is loaded in Chrome.

Step 2: Open Developer Tools

Press F12 or right-click and select "Inspect" to open Chrome DevTools.

Go to the Console tab to see logs.

Step 3: Open the Side Panel

Click the Shopping Assistant extension icon in the toolbar to open the side panel.

Step 5: Check Console Logs

Look for these log messages in the console:

  • Background: Starting voice recording...
  • Audio chunk received: {size: ..., chunks: ...}
  • Recording stopped, processing audio...
  • Audio data received: {dataLength: ..., mimeType: ...}
  • Starting Cartesia transcription...
  • Transcription result: ...

Troubleshooting:

  • If you see "No valid Cartesia API key found", check that config.js is properly loaded
  • If recording doesn't start, check for permission errors in the console
  • If no audio chunks are received, ensure your microphone is working
  • Check all extension contexts: popup console, background service worker console, and offscreen document console
